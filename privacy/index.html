プライバシーポリシー

施行日: 2026年7月3日（バージョン 1.2.0、最終更新: 2026年7月9日）

一言でまとめると: 体重の実数はあなたの端末内にのみ保存し、外部に送信しません。仲間に共有されるのは「体重の増減」「記録したかどうか」「あなたが宣言した目標」「表示名」「応援スタンプのやり取り」だけで、それも同じグループのメンバーに限られます。広告表示・トラッキングは行いません。なお、アプリの改善のために、起動やスタンプ送信などの操作の記録（体重の値は含みません）をサーバーに記録します。

1. はじめに

本プライバシーポリシーは、痩せないと（以下「本アプリ」）における利用者の情報の取り扱いについて説明します。本アプリは、利用者本人が招待した親しい友達・家族だけで作る非公開のグループ内で、ダイエットの記録を共有して励まし合うためのアプリです。体重の実数は端末内にのみ保存し、グループ内で共有されるのは増減・記録の有無・目標・表示名・応援スタンプのやり取りに限られます。

2. 収集・保存する情報

2.1 端末内にのみ保存される情報（外部送信なし）

以下の情報は利用者の端末内にのみ保存され、当社を含むいかなる外部サーバーにも送信されません。

  • 体重の実数値（記録日時を含む）
  • 体重を非表示にするかどうかの表示設定

なお、利用規約・プライバシーポリシーおよび記録共有への同意は、その場の画面操作で確認するものであり、同意の記録（同意日時・バージョンなど）を端末・サーバーのいずれにも保存しません。

2.2 グループ参加に同意した場合にサーバーへ送信・共有される情報

本アプリはグループでの記録共有機能を提供するため、以下の情報のみをクラウドサーバー（下記4に記載）へ送信し、同じグループのメンバーに共有します。自分の記録を共有する「主役（ダイエットする人）」として参加・作成する際には「記録を仲間に共有することへの同意」を必須としており、同意しない限り記録の送信・共有は行われません（記録を共有しない「見守る人」として参加する場合、共有対象となる記録はありません）。

  • 体重の増減値（前回・スタート比などの差分。実数そのものは送信しません）
  • 記録の有無（今日記録したか、直近いつ記録したか）
  • 自分で宣言した目標（目標増減、目標達成日）
  • 表示名
  • 応援スタンプ・定型メッセージ（誰から誰へ、どのスタンプを送ったか）

このほか、スタンプの受信設定（煽り系・悪魔スタンプを受け取るかどうかの ON/OFF）をサーバーに保存します。この設定は受け取りたくないスタンプを届く前に止めるために使われるもので、他のメンバーに公開はされません。

2.3 明示的に収集・利用しない情報

  • 体重の実数値は外部に送信しません。共有されるのは増減のみです。
  • 広告 ID（IDFA）の取得、広告配信、トラッキングは行いません。
  • 精密な位置情報（GPS）は取得しません。
  • 本アプリの情報を広告目的・第三者への販売目的で利用することはありません。

2.4 サービスの提供・改善のために記録する利用状況

本アプリの継続的な改善のため、以下の「操作の記録」を匿名の認証IDおよび日時とともにサーバーへ記録します。

  • アプリを起動したこと（1日1回）
  • ホーム画面を表示したこと（1日1回）
  • 応援スタンプを送信したこと

記録するのは上記の操作の種別と日時のみで、体重の実数値・増減値その他の健康情報は含みません。この情報は、利用者がアプリを継続して使えているかを把握し、機能を改善する目的にのみ利用し、広告や第三者への提供には利用しません。

3. アカウント登録について

本アプリはメールアドレス・パスワード等の入力なしでご利用いただけます。利用者の識別には匿名の認証 ID を使用し、氏名・メールアドレス・電話番号等の個人を直接特定する認証情報は取得しません。

4. 情報の保存場所と第三者提供・越境移転（個人情報保護法第28条関連）

第2.2 節に記載の共有情報は、本アプリのクラウド基盤上に保存され、本人が参加に同意したグループのメンバーにのみ共有されます。基盤の運用のため以下の国外事業者を利用します。個人情報保護法第28条（外国にある第三者への提供）に基づき、提供先・所在国を明示します。

  • Supabase Inc.（米国）: 本アプリのデータベース・認証基盤として、第2.2 節の共有情報および匿名認証 ID を保存します。同社の安全管理措置に基づき取り扱われます。
  • Apple Inc.（米国）: App Store・TestFlight を通じた配信について、Apple のプライバシーポリシーに従って取り扱われます。

これらの事業者への提供は、本アプリの記録共有機能を提供する目的に限られます。広告・解析目的での提供は行いません。

5. ユーザーの権利（確認・削除・退会）

  • 記録の削除: 端末内の体重記録は、本アプリをアンインストールすると端末内のデータごとすべて削除されます。アカウントを削除した場合も、端末内の記録を削除します。
  • グループからの退会: グループを退会することで、以降の記録共有を停止できます。
  • 受け取りたくない応援スタンプ: 設定でスタンプの受信をオフにできます。
  • アカウントの削除: アプリ内の「設定」→「アカウントを削除」から、いつでもアカウントとサーバー上の記録を削除できます（安全のために保存している通報の記録は、個人を特定できない形で残ることがあります）。
  • お問い合わせによる削除依頼: サーバーに保存された共有情報の削除をご希望の場合は、下記8の連絡先までご連絡ください。

6. 児童のプライバシー

本アプリは児童向けに特化して設計されたものではありません。未成年の方は、必要に応じて保護者の同意または監督のもとでご利用ください。本アプリは児童から個人情報を直接収集することはありません。

7. 医療判断ではないことの確認

本アプリは体重の記録と仲間との共有をサポートするツールであり、医学的診断や治療、栄養指導を提供するものではありません。健康管理上の判断は、医師や管理栄養士等の専門家にご相談いただくことを推奨します。
